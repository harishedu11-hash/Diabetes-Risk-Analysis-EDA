Project 1: Exploratory Data Analysis (EDA) of Diabetes Indicators
🩺 Overview
This project analyzes a dataset of ~250k patients to identify the primary lifestyle and clinical indicators of Diabetes and Pre-Diabetes.

📊 Key Findings (So Far)
Data Quality: The dataset is clean with no missing values.

Critical Insight: General Health (GenHlth) and High Blood Pressure (HighBP) show the strongest correlation with Diabetic status.

BMI Impact: There is a clear upward trend in median BMI from the Healthy group to the Diabetic group.

🛠️ Tools Used
Python (Pandas, Seaborn, Matplotlib)
